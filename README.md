# log_parser

あらゆるログを解析する

[![CI](https://github.com/koba-masa/log_parser/actions/workflows/ci.yml/badge.svg)](https://github.com/koba-masa/log_parser/actions/workflows/ci.yml)

## コマンド

- パッケージインストール

```sh
make install
```

- 実行

```sh
make run
```

- Linter

```sh
make run
```

- Lint(fix)

```sh
make fix
```

- テスト

```sh
make test
```

- CI

```sh
make ci
```
